あとがき

　折角なので白状しますが、人を殺してみたいです。嘘です。寧ろ殺されてみたいです。作中のエナのように可愛く一見弱そうで、それでも勝気な女の子に斬り殺されたいです。嘘です。でもフルボッコくらいにはされたいかもしれません。これは本当です。自分ドＭですので。

　そんなわけで本書をご覧の皆様こんにちは、 野村 周平 ( のむら しゅうへい ) と申します。普段は『 Ｍｃ ( まく ) 』と言うペンネームで、主にプログラマ兼シナリオライターとして東方プロジェクトの二次創作ゲーム制作に携わったり、時折ボーカロイドのイラストを描いたりと言った活動をしてますが、今回初の中・長編規模のオリジナル小説を書いてみました。数行から十ページ程度の所謂ショート・ショートなら幾度か書いたことがあるのですが、この規模のものは正真正銘初めてですね。

　このシナリオの原案は十五歳頃、他人より一足遅れて中二病が発動し出した頃、夢の中に出てきたのが最初でした。そして翌年には話の大筋が完成、当初これを漫画化する予定でしたが、諸々あって四分の一程度描いた辺りで挫折しました。それ以来長いこと忘れていたのですが、自宅の蔵書数が増えてきて（自炊大嫌い人間なので）、整理も兼ねて集密書架もどきを自作している際に当時のネームノートを発見し、今一度書いてみようかと思い立って、この度完成した次第です。今から思い返すと、自分文章書くのが大の苦手なのに良くここまで書けたものだ、と自分自身に驚いています。小中学生時代の読書感想文などの各種作文は、八割近くが親の代筆と言う大変ダメな子でした。ところでまさか居ないとは思いますが、高校生当時途中まで描いていた漫画を知っている方、当時とキャラデザインがかなり違うことにお気付きかと思います。実はそれから数年後、今からだと十一年程前に頒布された、とあるゲームの主人公たちをちょっとだけリスペクトしています。自分は当時からそのゲームの大ファンでしたので、もしかしたらここでぶっちゃける前に気付いてしまった方も居るかもしれませんね。それだけにこんな結末ではなくて、もう少し幸せに終わらせてあげたかったのですが、もにょもにょ……。

　さて次回ですが、またプログラマに戻り東方二次創作ゲームを作っていく予定です。この作品では殆ど描写しきれていない設定……グレース幼少期のヘタレ成長記とか、青年期のギルドで殺戮に明け暮れた話とか、或いはバルマン領主が青年期に首都の迫害を受けて独り身となった悲劇の物語とかを、続編または外伝としていつか書いてみたい気持ちはあるのですが、正直なところなんだかんだ言ってやっぱり作文は不得手なので、やるかどうかは微妙ですね。

　最後に、今回本書のイラスト全般を手掛けてくださった絵師の桜姫つかささん、大変素敵な絵師さんを紹介してくださったスタジオホビーの白鳥さん（学生時代の旧友です）、制作中様々なご指摘、ご意見を戴いた方々、そして何と言ってもここまで読んでくださった皆様にこの場を借りてお礼申し上げます。それではまた次回作でお会いしましょう。

脱稿までに一度ハルツ地方の集落とか散策したかった、著・野村 周平（Ｍｃ）
